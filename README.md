# Crime Analysis Dashboard of India (2020–2024)

## 📌 Project Overview

The Crime Analysis Dashboard is an interactive Power BI project designed to analyze crime patterns across India from 2020 to 2024. The dashboard provides insights into crime trends, geographical distribution, victim demographics, weapon usage, and case closure status to support data-driven decision-making.

## 🎯 Objectives

- Analyze crime occurrences across different states and cities.
- Identify crime hotspots using geographical visualization.
- Examine victim demographics such as age and gender.
- Study weapon usage in criminal activities.
- Monitor case closure and investigation status.
- Provide an interactive platform for crime trend analysis.

## 🛠️ Tools & Technologies Used

- Power BI
- Microsoft Excel
- Data Cleaning & Transformation
- Data Visualization
- DAX (Data Analysis Expressions)

## 📊 Dashboard Pages

### 1. Main Dashboard
Provides an overview of:
- Total Crime Cases

- Case Status Distribution

### 2. Locality-Based Analysis
Provides insights into:

- City-wise Crime Analysis

### 3. Type Exploration Dashboard
Provides analysis of:
- Crime Categories
- Weapon Usage
- Victim Demographics
- Crime Domain Distribution
- Crime Type Trends
